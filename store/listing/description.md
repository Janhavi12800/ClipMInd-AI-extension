# TechShield AI — Full Store Description

## Your All-in-One AI Security & Productivity Companion

TechShield AI is the enterprise-grade Chrome extension that combines AI-powered prompt generation, real-time website security scanning, SEO analysis, and productivity tools — everything professionals need to browse, work, and create with confidence.

---

### 🤖 AI Prompt Tools
- **Prompt Generator** — 6 enterprise templates for content, security reports, marketing, and development
- **Prompt Enhancer** — Transform rough ideas into polished, professional prompts
- **Prompt Library** — Save, organize, and reuse your best prompts across sessions

### 🛡️ Security Suite
- **Website Security Scanner** — Analyze any page for vulnerabilities, mixed content, and insecure headers
- **Phishing Detector** — Real-time typosquatting, suspicious TLD, and credential harvesting detection
- **Malware Warning** — Alerts for suspicious scripts, auto-downloads, and malicious URL patterns

### 📊 SEO & Technology Analysis
- **SEO Analyzer** — Overall page SEO scoring with actionable recommendations
- **Meta Tag Analyzer** — Title, description, canonical, and Open Graph validation
- **Heading Structure** — H1–H6 hierarchy analysis for accessibility and SEO
- **Technology Detector** — Identify CMS, frameworks, analytics, and 30+ technologies

### 📝 Productivity
- **Notes Manager** — URL-anchored notes with tags for research and client work
- **Productivity Toolkit** — Text snippets, task lists, and Pomodoro timer

### Permissions Explained

TechShield AI requests permissions only to deliver its features:
- **Active tab access** — To analyze the page you're viewing (only when you click Scan)
- **Storage** — To save your prompts, notes, and settings
- **Notifications** — To alert you about security threats

We never sell your data. Read our full Privacy Policy at https://techshield.ai/privacy
